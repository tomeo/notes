# Show execution plan MSSQL

```
SET STATISTICS PROFILE ON
<Query>
SET STATISTICS PROFILE OFF
```

## Other options
* `SET SHOWPLAN_TEXT ON`
* `SET SHOWPLAN_ALL ON`
* `SET SHOWPLAN_XML ON`
* `SET STATISTICS PROFILE ON`
* `SET STATISTICS XML ON`
